## 中文社区短帖

我把最近开发文档 skill 时遇到的问题整理成了一个开源项目：Agent Doc Generator。

用 Agent 写代码很方便，开发文档却经常要重新整理一遍。对不熟悉文档脚本的人来说，截图、流程图、中文乱码和图片损坏都很容易卡住。

这个 skill 会让 Agent 自动完成几件事：

- 从开发对话、项目笔记和仓库内容整理文档结构
- 用 Playwright 截取真实界面
- 用代码渲染带准确文字的流程图和架构图
- 在 Codex 中调用 `imagegen` 补充概念解说图
- 把中文文本和图片二进制分开处理，避免编码错误
- 同时交付 DOCX 和 `gen_doc.js`，后续可以修改和重跑

仓库里放了一份自举案例：这个 skill 用自己生成了自己的开发文档，截图计划、图片清单、生成脚本和最终 Word 文件都可以直接检查。

## 发布顺序

1. 先发布 GitHub 仓库，确认安装命令和 Actions 通过。
2. 发布一条带流程图和自举 DOCX 的中文短帖。
3. 发布 X 帖子，首句说明产物，第二段放仓库链接以生成预览。
4. 收集首批用户遇到的安装和截图问题，补充 README。
5. 有两个以上真实使用案例后，再发布 Show HN 和长文复盘。

## skills.sh 收录

不需要提交申请。仓库公开后，让用户运行：

```bash
npx skills add shenyuhao1/agent-doc-generator --skill agent-doc-generator
```

Skills CLI 的匿名安装遥测会让仓库自动进入 skills.sh，并按安装量参与排行。CI 环境默认不发送遥测，因此发布后需要在正常终端中完成至少一次真实安装。
