Agent Doc Generator 图片流水线

先准备可追溯的图片资产，再用稳定 ID 写入 DOCX

images.json / v1

现有图片

复用项目中的截图、图表、Logo 和已有设计稿。

真实界面截图

Playwright 打开网页，等待目标状态后截取整页或指定元素。

精确图解

用 HTML、CSS、SVG 或 Mermaid 绘制，保证标签和结构准确。

Codex 生图

生成概念插画或教学示意图，复制到工作区后进入同一清单。

↓
准备资产 prepare-images.mjs

复制文件、自动截图、检测 PNG/JPEG 尺寸，并写入 UTF-8 清单。

→
统一清单 doc-assets/images.json

记录稳定 ID、相对路径、尺寸、题注、替代文本、来源和必需状态。

→
写入 DOCX imageBlock("pipeline")

图片以二进制 Buffer 进入 ImageRun，按页面尺寸等比缩放并添加题注。

图片字节始终保持二进制，不经过 UTF-8 解码、Base64 文本替换或 JSON 内联。
必需图片缺失时直接停止生成，避免文档悄悄漏图或用错误图片替代。
